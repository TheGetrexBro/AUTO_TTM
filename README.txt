---УСТАНОВКА---
1. Сначала скачайте и установите интерпретатор Python
Сайт: python.org/downloads
Важно: Во время установки поставьте галочку Add Python to PATH (это позволит запускать Python из командной строки).

2. После установки Python откройте командную строку (Win + R -> cmd) и введите команды (после каждой нажимайте Enter):
pip install keyboard
pip install pyperclip
pip install tkinter

3. Введите в командной строке:
python -c "import tkinter; print('Готово!')"
Если видите сообщение "Готово!" — всё установлено корректно.

---ИСПОЛЬЗОВАНИЕ---
1. Откройте двумя кликами лкм start_ttm_app.bat
2. Настройте приложение под концерт, который вы желаете
3. Откройте страницу концерта и нажмите "BUY NOW"
4. Когда у вас открылась страница Terms & Conditions нажмите F12 и нас интересует вкладка консоль, с которой мы в дальнейшем будем рабоать
5. Нажмите горячие клавиши, которые вы выбрали в настройках и далее нажмите ctrl + v и нажмите enter и так на всех этапах

---ВНИМАНИЕ----
1. Первый этап иногда может не появлятся, поэтому, если у вас так и произошло, то еще раз нажмите на горячую клавишу и вы перейдете на следующий этап
2. Если у вас автоматически уже стоит нужная дата концерта, то просто еще раз нажмите на горячую клавишу и вы перейдете на следующий этап
3. Если возникнут вопросы, задавайте спокойно мне
